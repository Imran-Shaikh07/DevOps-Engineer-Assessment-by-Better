About Task Tracker

This is a minimal, modern Flask web application for tracking tasks. It is designed for easy deployment and demonstration in CI/CD pipelines, and is suitable for platforms like Render.

  • All tasks are stored in memory (reset on restart).
  • Built with Python, Flask, and Jinja2 templates.
  • Modern responsive design with a simple UI.

Created for DevOps and deployment demos.
